# Available At Section - Design Concepts

Here are 3 distinct design concepts for the new **"Available At / Follow Us"** section. These designs preserve the **Chhattisgarhi Retro Zine / Neo-brutalist** theme (yellow `#FACC15`, off-black/charcoal, thick borders, offset shadows) and offer creative, high-fidelity representations of streaming platforms and social media.

---

## 🎚️ Concept 1: "The Radio Dial & Frequency Tuner" (Console Preset Panel)
*Inspired by the dial scale and preset push-buttons of a vintage FM/AM receiver or car radio console. Platforms are mapped to radio frequencies.*

```
+--------------------------------------------------------------------------------+
|  STATION RECEIVER // TUNE IN TO THE RAIPUR PODCAST                             |
|                                                                                |
|  [ DIAL SCALE ]                                                                |
|  =====================(| SPOTIFY: 104.2 )===============(| YOUTUBE: 107.8 )===  |
|  | | | | | | | | | | | |       [o]       | | | | | | | | | |    [o]    | | | | |  |
|                                                                                |
|  [ PRIMARY STREAMS ]                                                           |
|  +-------------------------------------+ +-----------------------------------+ |
|  |  🎧 SPOTIFY PODCASTS                | |  📺 YOUTUBE VIDEO FEED            | |
|  |  Listen to full crystal audio.      | |  Watch full episode recordings.   | |
|  |                                     | |                                   | |
|  |  [ LAUNCH AUDIO PLAYER ⚡ ]         | |  [ TUNE IN TO YOUTUBE ⚡ ]        | |
|  +-------------------------------------+ +-----------------------------------+ |
|                                                                                |
|  [ SOCIAL TRANSMITTERS // PRESET DIAL BUTTONS ]                                |
|  +--------------+ +---------------+ +---------------+ +-----------------+      |
|  | (o) INSTA    | | (o) FACEBOOK  | | (o) THREADS   | | (o) PINTEREST   |      |
|  | 98.3 FM      | | 91.1 FM       | | 100.4 FM      | | 105.7 FM        |      |
|  +--------------+ +---------------+ +---------------+ +-----------------+      |
+--------------------------------------------------------------------------------+
```

* **Visual Elements**:
  * **Frequency Tuner Scale**: A horizontal tuning slider at the top. The "needle" indicator shifts dynamically to point to the active streaming logo when the user hovers over either Spotify or YouTube cards.
  * **Primary Streaming Cards**: Two large, side-by-side columns:
    * **Spotify**: Styled with a solid black backdrop, neon green border/shadow offset, and custom headphones badge.
    * **YouTube**: Styled with a solid black backdrop, crimson red border/shadow offset, and play-button graphics.
  * **Preset Push-Buttons**: The social networks (Facebook, Instagram, Threads, Pinterest) are styled as physical retro select-buttons on a vintage car dashboard. Pushing a button shifts it downward (`active:translate-y-1 active:shadow-none`) and opens the link.
* **Why it works**: Clean, highly cohesive with the radio theme, and gamifies clicking on social media.

## 🎨 Concept 2: "The Gig Poster / Sticker Collage" (Street Wall Vibe)
*Styled like a retro brick wall or cardboard texture background, pasted with overlapping wheat-paste concert gig flyers and sticker slaps.*

```
+--------------------------------------------------------------------------------+
|  STREET CHRONICLES // SCAN THE AREA                                            |
|                                                                                |
|     +-------------------------+         +--------------------------+           |
|     |  * GIG POSTER: SPOTIFY *|         |  * GIG POSTER: YOUTUBE * |           |
|     |   ===================== |         |   ====================== |           |
|     |   THE RAIPUR PODCAST    |         |   WATCH EXCLUSIVE FEEDS  |           |
|     |   Stream full episodes  |         |   High-def video logs.   |           |
|     |   anytime, anywhere.    |         |                          |           |
|     |                         |         |      [ WATCH ON YT ]     |           |
|     |    [ LISTEN ON SPOTIFY] |         +--------------------------+           |
|     +-------------------------+                                                |
|                                         +------------------+                   |
|       +----------+    +-----------+     | STICKER: INSTA   |   +------------+  |
|       | FB SLAP  |    | THREADS   |     | Follow daily!    |   | PINTEREST  |  |
|       +----------+    +-----------+     +------------------+   +------------+  |
+--------------------------------------------------------------------------------+
```

* **Visual Elements**:
  * **Asymmetrical Overlap**: The posters are slightly rotated (`-rotate-1` and `rotate-1`).
  * **Sticker Slaps**: Social links are styled like round or rectangular adhesive stickers with die-cut white borders, halftone color patterns, and rough edges.
  * **Tactile Hover**: Hovering on a poster or sticker straightens it out (`hover:rotate-0 hover:scale-105`) and increases its offset shadow, making it feel like it's peeling off the page.
* **Why it works**: Incredibly stylized, zine-like, and gives the app an indie, underground culture feeling that matches the Chhattisgarhi street stories.

## 📻 Concept 3: "The Cassette Jukebox Shelves" (Record Store Bin)
*Organizes streaming options as premium physical items sitting in a wooden or steel record store slot shelf.*

```
+--------------------------------------------------------------------------------+
|  RECORD STORE STAND // COLLECT OUR RELEASES                                    |
|                                                                                |
|   +-------------------------------------++-----------------------------------+ |
|   | [ CASSATE TAPE: SPOTIFY RELEASE ]   || [ VIDEO CASSETTE: YOUTUBE FEED ]  | |
|   |  +-------------------------------+  ||  +------------------------------+ | |
|   |  |   SPOTIFY TAPE [========]     |  ||  |   YOUTUBE VHS  |  ( ) ( ) |  | | |
|   |  |   * Click to pull forward *   |  ||  |   * Click to pull forward *  | | |
|   |  +-------------------------------+  ||  +------------------------------+ | |
|   +-------------------------------------++-----------------------------------+ |
|                                                                                |
|   [ SOCIAL MEDIA CATALOG CARDS ]                                               |
|   +---------------+ +---------------+ +---------------+ +------------------+   |
|   | CARD: FB      | | CARD: INSTA   | | CARD: THREADS   | | CARD: PINTEREST|   |
|   | (Folded card) | | (Folded card) | | (Folded card) | | (Folded card)  |   |
|   +---------------+ +---------------+ +---------------+ +------------------+   |
+--------------------------------------------------------------------------------+
```

* **Visual Elements**:
  * **Physical Shelf Grids**: The Spotify and YouTube options are housed in cassettes or VHS tape sleeves.
  - Clicking/hovering slides the tape cassette up out of its cardboard sleeve, revealing a "Tune In" badge.
  * **Folded Catalog Cards**: Social links are styled as folded paper cards inside small archive drawers.
* **Why it works**: Retro cassette-collector vibes that complement the tape cabinet section.
